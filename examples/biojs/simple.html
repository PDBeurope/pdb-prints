PDBPrints Demo

I) Basic usage



<pdb-prints pdb-ids="['1cbs']"></pdb-prints>


II) Using 'settings' attribute



<pdb-prints pdb-ids="['1cbs']" settings='{"orientation": "horizontal", "size": 48 }'></pdb-prints>


III) Selective logo hiding - "hideLogo" option



<pdb-prints pdb-ids="['1cbs']" settings='{"size": 48, "hideLogo": ["PDBeLogo","PrimaryCitation"] }'></pdb-prints>

How to load multiple PDBPrints in a single call?
Add 'pdb-prints' and 'pdb-ids' attribute to the main(parent) <div> as shown in code snippet below.
To display it in a specific place, add <div> or <span> and specify its class as "pdbprints_'PDBID'" (example : "pdbprints_1cbs").
If there is no element with class "pdbprints_'PDBID'" then they are appended to the main(parent) <div> by default. Check out the code snippet below : 

<div pdb-prints pdb-ids="['1cbs','1tqn']">
  <div class="proteinSection">
    <h5>PDBPrints for 1CBS</h5>
    <div class="pdbprints_1cbs"></div>
  </div>
  <br><br>
  <div class="proteinSection">
    <h5>PDBPrints for 1TQN</h5>
    <div class="pdbprints_1tqn"></div>
  </div>
</div>
